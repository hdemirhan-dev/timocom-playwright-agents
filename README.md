# timocom-playwright-agents

Playwright test suite for timocom.de, fully generated and maintained by Playwright Agents (--loop=claude).

## Setup

```bash
npm install
npx playwright install chromium
```

Set your API key:
```bash
export ANTHROPIC_API_KEY=your_key_here
```

---

## Step 1 — Run the Planner (Haiku, ~$0.05–0.10)

Open Claude Code in this project folder, select the **playwright-test-planner** agent, and send:

```
Use tests/seed.spec.ts as the seed file.

Navigate to https://www.timocom.de and explore only these pages:
1. Homepage (https://www.timocom.de)
2. Registration (https://www.timocom.de/registrierung)
3. Services (https://www.timocom.de/services)

Save the test plan as specs/timocom-plan.md

STRICT rules:
- Max 4 browser_snapshot calls total — one per page
- Do NOT take screenshots
- Do NOT follow language switcher links
- Do NOT explore login (my.timocom.com) — out of scope
- Cookie banner: call UC_UI.acceptAllConsents() via browser_evaluate
- AEM double-rendered elements → note .first() in steps
- Never reference #ci-NNNNN IDs
- Max 3 test scenarios per section, max 6 steps each
```

---

## Step 2 — Run the Generator (Sonnet, ~$0.20–0.35)

Switch to **playwright-test-generator** agent and send:

```
Read specs/timocom-plan.md and generate Playwright TypeScript tests for all scenarios.
Use tests/seed.spec.ts as the seed file.

Generate one file per section:
- tests/homepage.spec.ts
- tests/registration.spec.ts  
- tests/services.spec.ts

AEM rules to apply in every test:
- beforeEach: goto timocom.de → waitForFunction UC_UI defined → acceptAllConsents() → waitForFunction isInitialized()
- Double-rendered elements → .first()
- Never use #ci-NNNNN IDs
- Use getByRole / getByText / getByLabel — no CSS class selectors
- Navigation: page.goto(url) — not .click() on nav links
```

---

## Step 3 — Run the Healer (Sonnet, ~$0.10–0.20)

Switch to **playwright-test-healer** agent and send:

```
Run all tests in the tests/ folder, debug failures, and fix them.
Apply AEM rules: UC_UI cookie setup in beforeEach, .first() for double-rendered elements,
replace any #ci-NNNNN IDs with stable getByRole locators.
Continue until all tests pass or are marked test.fixme() with explanation.
```

```bash
npx playwright test
npx playwright test --headed        # watch the browser
npx playwright show-report          # open HTML report
```
