# timocom-playwright-agents

Playwright test suite for timocom.de, fully generated and maintained by Playwright Agents (--loop=claude).

**Branch: `explore/frachtenboerse`** — this branch targets only the public Frachtbörse page (`/services/frachtenboerse`), as a fresh, from-scratch exploration pass in plain JavaScript.

## Setup

```bash
npm install
npx playwright install chromium
```

Set your API key:
```bash
export ANTHROPIC_API_KEY=your_key_here
```

---

## Step 1 — Run the Planner (Haiku, ~$0.05–0.10)

Open Claude Code in this project folder, select the **playwright-test-planner** agent, and send:

```
Use tests/seed.spec.js as the seed file.

Navigate to https://www.timocom.de/services/frachtenboerse and explore this page thoroughly.

Identify all interactive elements, forms, navigation paths, and functionality.
Determine yourself whether a cookie consent banner appears and how to dismiss it.
Determine yourself whether any elements are duplicated in the DOM.

Save the test plan as specs/frachtenboerse-plan.md

Constraints:
- Max 2 browser_snapshot calls
- Do NOT take screenshots
- Do NOT navigate away from this page (no login, no language switch, no other services pages)
- Max 4 test scenarios, max 6 steps each
```

---

## Step 2 — Run the Generator (Sonnet, ~$0.10–0.20)

Switch to **playwright-test-generator** agent and send:

```
Read specs/frachtenboerse-plan.md and generate Playwright JavaScript tests (plain .js, NOT TypeScript) for all scenarios.
Use tests/seed.spec.js as the seed file.

Generate: tests/frachtenboerse.spec.js

Use resilient, user-facing locators (getByRole, getByLabel, getByText) verified against the live page.
Use CommonJS require() syntax, matching tests/seed.spec.js.
```

---

## Step 3 — Run the Healer (Sonnet, ~$0.10–0.20)

Switch to **playwright-test-healer** agent and send:

```
Run tests/frachtenboerse.spec.js, debug any failures, and fix them.
Continue until all tests pass or are marked test.fixme() with an explanation of the underlying issue.
```

## Run tests manually

```bash
npx playwright test
npx playwright test --headed        # watch the browser
npx playwright show-report          # open HTML report
```
